# SQL Practice Repository

## Description
This repository contains SQL scripts for practice and learning purposes. It demonstrates the use of various SQL concepts including:

- Database creation and selection
- Table creation with constraints (PRIMARY KEY, CHECK, NOT NULL, ENUM)
- Data insertion (INSERT)
- Data retrieval (SELECT, WHERE, BETWEEN, IN, LIKE)
- Data modification (UPDATE, DELETE)
- Table alteration (ALTER TABLE: ADD, MODIFY, DROP columns)
- Conditional logic (CASE)
- Aggregation functions (COUNT, AVG, MAX, MIN)
- Grouping and filtering data (GROUP BY, HAVING)
- Ordering and limiting results (ORDER BY, LIMIT)

## Files
- `School_Employee_Assignment.sql` — Covers SCHOOL and EMPLOYEE databases with multiple SQL operations  
- `College_Performance_Tracker.sql` — Covers a college student performance tracker with marks, grades, and aggregations

## Purpose
To provide practical examples and exercises for learning SQL concepts and database management.
